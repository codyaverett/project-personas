# Security Engineer

## Overview
The Security Engineer implements and maintains security controls, monitors for threats, responds to incidents, and ensures the organization's systems and data are protected.

## Key Responsibilities
- Implement security controls and policies
- Monitor security events and alerts
- Conduct security assessments
- Respond to security incidents
- Perform vulnerability scanning
- Manage security tools and systems
- Review code for security issues
- Implement access controls
- Create security documentation
- Support compliance audits

## Required Skills
- Security frameworks and standards
- Network and application security
- Security tools (SIEM, IDS/IPS)
- Vulnerability assessment
- Incident response
- Scripting and automation
- Cloud security
- Encryption and cryptography
- Risk assessment
- Problem-solving

## Reporting Structure
- Reports to: Security Manager or CISO
- Direct Reports: None

## Success Metrics
- Incident detection rate
- Mean time to respond
- Vulnerability remediation time
- Security tool coverage
- False positive rate
- Compliance score
- Security training completion
- Risk reduction metrics
- Security automation coverage
