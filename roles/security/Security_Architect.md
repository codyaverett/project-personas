# Security Architect

## Overview
The Security Architect designs secure systems and infrastructure, establishes security standards, and ensures security is built into all technology solutions.

## Key Responsibilities
- Design security architectures
- Define security requirements
- Review system designs for security
- Establish security patterns and standards
- Evaluate security technologies
- Lead threat modeling exercises
- Design identity and access management
- Develop security reference architectures
- Guide secure development practices
- Assess third-party security

## Required Skills
- Security architecture frameworks
- Threat modeling
- Secure design principles
- Cloud security architecture
- Identity management
- Cryptography
- Network security
- Application security
- Risk assessment
- Communication

## Reporting Structure
- Reports to: CISO or Head of Security
- Direct Reports: None (technical leadership)

## Success Metrics
- Architecture review coverage
- Security design pattern adoption
- Vulnerability prevention rate
- Security requirement compliance
- Threat model coverage
- Security control effectiveness
- Architecture documentation quality
- Security review turnaround
- Risk reduction impact
