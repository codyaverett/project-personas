# Senior QA Engineer

## Overview
The Senior QA Engineer leads quality initiatives, develops test strategies, mentors junior QA engineers, and drives test automation efforts across products.

## Key Responsibilities
- Develop comprehensive test strategies
- Lead test automation initiatives
- Design complex test scenarios
- Mentor junior QA engineers
- Perform security and performance testing
- Review and improve QA processes
- Collaborate with architects on testability
- Lead test planning for releases
- Conduct risk assessments
- Drive quality metrics

## Required Skills
- Advanced testing expertise
- Test automation frameworks
- Performance testing
- Security testing
- Leadership and mentoring
- Process improvement
- Risk analysis
- Programming skills
- CI/CD knowledge
- Communication

## Reporting Structure
- Reports to: QA Manager or Head of QA
- Direct Reports: QA Engineers (mentoring)

## Success Metrics
- Test automation coverage
- Quality improvement metrics
- Release quality scores
- Process efficiency gains
- Team skill development
- Defect prevention rate
- Test strategy effectiveness
- Automation ROI
- Production incident reduction
