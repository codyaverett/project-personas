# Tech Lead

## Overview
The Tech Lead combines hands-on technical work with leadership responsibilities, guiding technical decisions and coordinating the team's technical efforts.

## Key Responsibilities
- Lead technical planning and design
- Guide architectural decisions
- Coordinate technical work across the team
- Provide hands-on coding and development
- Mentor team members technically
- Facilitate technical discussions
- Ensure code quality and standards
- Manage technical debt
- Interface with stakeholders on technical matters
- Drive technical innovation

## Required Skills
- Strong technical skills
- Leadership and influence
- Architecture and design
- Communication skills
- Problem-solving
- Team coordination
- Code review expertise
- Project planning
- Mentoring abilities
- Decision-making

## Reporting Structure
- Reports to: Engineering Manager or Director
- Direct Reports: None (leads technically)

## Success Metrics
- Technical delivery success
- Code quality improvements
- Architecture decisions
- Team technical growth
- Technical debt management
- Innovation initiatives
- Documentation quality
- Technical standards adoption
- Project technical success
