# Full Stack Developer

## Overview
The Full Stack Developer works across both frontend and backend technologies, building complete features from user interface to database, with expertise in multiple technology stacks.

## Key Responsibilities
- Develop frontend user interfaces
- Build backend APIs and services
- Design and optimize databases
- Implement full feature sets end-to-end
- Write responsive and accessible code
- Integrate third-party services
- Debug across the entire stack
- Optimize application performance
- Collaborate with designers and product
- Maintain code quality standards

## Required Skills
- Frontend technologies (HTML, CSS, JavaScript, React/Vue/Angular)
- Backend languages (Node.js, Python, Java, etc.)
- Database systems (SQL and NoSQL)
- RESTful API design
- Version control (Git)
- Responsive design
- Testing frameworks
- Cloud services basics
- Problem-solving
- Communication

## Reporting Structure
- Reports to: Engineering Manager or Tech Lead
- Direct Reports: None

## Success Metrics
- Feature completion rate
- Code quality metrics
- Cross-stack proficiency
- Bug rate
- Performance improvements
- Code review effectiveness
- Learning and adaptation
- Team collaboration
- Documentation quality
