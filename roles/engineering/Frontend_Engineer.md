# Frontend Engineer

## Overview
The Frontend Engineer specializes in building user interfaces and client-side applications, focusing on user experience, performance, and modern web technologies.

## Key Responsibilities
- Develop responsive web applications
- Implement UI/UX designs
- Optimize frontend performance
- Ensure cross-browser compatibility
- Write maintainable JavaScript/TypeScript
- Implement state management
- Build reusable components
- Integrate with backend APIs
- Conduct frontend testing
- Ensure accessibility standards

## Required Skills
- JavaScript/TypeScript expertise
- Modern frameworks (React, Vue, Angular)
- HTML5 and CSS3
- Responsive design
- State management (Redux, MobX)
- Build tools (Webpack, Vite)
- Testing (Jest, Cypress)
- Performance optimization
- Version control
- UI/UX principles

## Reporting Structure
- Reports to: Engineering Manager or Frontend Lead
- Direct Reports: None

## Success Metrics
- UI implementation accuracy
- Performance metrics (Core Web Vitals)
- Code quality and maintainability
- Component reusability
- Accessibility compliance
- Browser compatibility
- Bug rates
- Feature delivery speed
- Test coverage
