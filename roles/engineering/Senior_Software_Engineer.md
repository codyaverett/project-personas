# Senior Software Engineer

## Overview
The Senior Software Engineer designs and develops complex software systems, provides technical leadership, mentors junior engineers, and drives technical excellence within the team.

## Key Responsibilities
- Design and implement complex software features
- Lead technical design and architecture decisions
- Write high-quality, maintainable code
- Conduct code reviews and provide feedback
- Mentor junior and mid-level engineers
- Collaborate with product and design teams
- Investigate and resolve complex technical issues
- Contribute to technical documentation
- Participate in on-call rotations
- Drive best practices and standards

## Required Skills
- Advanced programming skills
- System design and architecture
- Problem-solving and debugging
- Code review and mentoring
- Communication and collaboration
- Testing and quality assurance
- Performance optimization
- Security best practices
- Version control (Git)
- Agile development

## Reporting Structure
- Reports to: Engineering Manager or Tech Lead
- Direct Reports: None (may mentor junior engineers)

## Success Metrics
- Code quality and maintainability
- Feature delivery and complexity
- Code review effectiveness
- Mentorship impact
- Technical documentation
- Bug resolution rate
- Performance improvements
- Knowledge sharing contributions
- On-time delivery
