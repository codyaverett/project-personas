# UI Designer

## Overview
The UI Designer creates visually appealing and functional interfaces, translating wireframes into beautiful designs while maintaining brand consistency and usability.

## Key Responsibilities
- Design user interfaces for web and mobile
- Create visual design systems and style guides
- Develop high-fidelity mockups and prototypes
- Ensure brand consistency across products
- Collaborate with UX designers and developers
- Create icons, illustrations, and graphics
- Design responsive layouts
- Maintain design libraries
- Implement design feedback
- Stay current with design trends

## Required Skills
- Visual design principles
- Typography and color theory
- Design software mastery
- Responsive design
- Design systems
- Prototyping tools
- Attention to detail
- Creative thinking
- Communication
- Time management

## Reporting Structure
- Reports to: Head of Design or Design Lead
- Direct Reports: None

## Success Metrics
- Design quality and consistency
- Design system compliance
- Delivery timelines
- Developer handoff efficiency
- Visual appeal ratings
- Brand consistency
- Design revision cycles
- Asset library maintenance
- Stakeholder satisfaction
