# UX Designer

## Overview
The UX Designer creates user-centered designs by understanding business requirements, user needs, and technical constraints to deliver intuitive and effective user experiences.

## Key Responsibilities
- Conduct user research and usability testing
- Create user personas and journey maps
- Design wireframes and prototypes
- Develop information architecture
- Collaborate with product and engineering
- Create user flows and interaction designs
- Conduct competitive analysis
- Iterate based on user feedback
- Ensure accessibility standards
- Document design decisions

## Required Skills
- User research methodologies
- Wireframing and prototyping
- Usability testing
- Information architecture
- Interaction design
- Design tools (Figma, Sketch, etc.)
- User psychology
- Problem-solving
- Communication
- Collaboration

## Reporting Structure
- Reports to: Head of Design or Design Director
- Direct Reports: None

## Success Metrics
- User satisfaction scores
- Usability test results
- Task completion rates
- Design iteration efficiency
- Accessibility compliance
- User engagement metrics
- Design system adoption
- Time to design completion
- Stakeholder satisfaction
