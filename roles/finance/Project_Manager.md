# Project Manager

## Overview
The Project Manager plans, executes, and delivers projects on time and within budget, coordinating resources, managing stakeholders, and ensuring project success.

## Key Responsibilities
- Define project scope and objectives
- Create project plans and schedules
- Manage project resources and budget
- Coordinate cross-functional teams
- Track progress and milestones
- Identify and mitigate risks
- Communicate with stakeholders
- Ensure quality deliverables
- Manage project documentation
- Conduct project retrospectives

## Required Skills
- Project management methodologies (PMP, Agile)
- Planning and organization
- Risk management
- Budget management
- Communication and presentation
- Leadership and influence
- Problem-solving
- Time management
- Stakeholder management
- Tools (Jira, Asana, MS Project)

## Reporting Structure
- Reports to: Program Manager or PMO Director
- Direct Reports: None (matrix management)

## Success Metrics
- On-time delivery rate
- Budget adherence
- Scope creep management
- Stakeholder satisfaction
- Risk mitigation effectiveness
- Resource utilization
- Quality metrics
- Team satisfaction
- Project ROI
