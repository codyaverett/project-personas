# Cloud Architect

## Overview
The Cloud Architect designs and oversees cloud computing strategies, ensuring scalable, secure, and cost-effective cloud infrastructure solutions.

## Key Responsibilities
- Design cloud architecture solutions
- Define cloud strategy and roadmap
- Evaluate and select cloud services
- Ensure security and compliance
- Optimize cloud costs
- Lead cloud migrations
- Design disaster recovery solutions
- Establish cloud governance
- Provide technical guidance
- Stay current with cloud technologies

## Required Skills
- Cloud platform expertise
- Architecture design patterns
- Security and compliance
- Cost optimization
- Networking and infrastructure
- Containerization and microservices
- DevOps practices
- Solution design
- Communication
- Strategic thinking

## Reporting Structure
- Reports to: CTO or VP of Engineering
- Direct Reports: None (technical leadership)

## Success Metrics
- Architecture scalability
- System reliability
- Cost optimization savings
- Security compliance
- Migration success rate
- Performance improvements
- Technology adoption
- Documentation quality
- Stakeholder satisfaction
