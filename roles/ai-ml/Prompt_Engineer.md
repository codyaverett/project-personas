# Prompt Engineer

## Overview
The Prompt Engineer specializes in designing, optimizing, and maintaining prompts for large language models, ensuring optimal AI system performance and user experience.

## Key Responsibilities
- Design effective prompts for LLMs
- Optimize prompt performance and accuracy
- Create prompt templates and libraries
- Test and iterate prompt variations
- Implement prompt chaining strategies
- Monitor prompt effectiveness
- Collaborate with product teams
- Document prompt best practices
- Train teams on prompt engineering
- Evaluate new LLM capabilities

## Required Skills
- LLM expertise (GPT, Claude, etc.)
- Natural language understanding
- Prompt design patterns
- A/B testing and experimentation
- Data analysis
- Technical writing
- API integration
- Python programming
- Critical thinking
- Communication skills

## Reporting Structure
- Reports to: AI Product Manager or Head of AI
- Direct Reports: None

## Success Metrics
- Prompt accuracy and reliability
- Response quality scores
- User satisfaction metrics
- Prompt efficiency (tokens used)
- Cost optimization
- Error rate reduction
- Template reusability
- Documentation quality
- Training effectiveness
