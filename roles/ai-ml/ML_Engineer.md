# Machine Learning Engineer

## Overview
The ML Engineer builds and deploys machine learning models at scale, focusing on the engineering aspects of ML systems including infrastructure, pipelines, and productionization.

## Key Responsibilities
- Design and build ML pipelines
- Deploy models to production
- Optimize model performance and scalability
- Build feature engineering pipelines
- Implement ML monitoring and observability
- Manage model versioning and experiments
- Collaborate with data scientists
- Ensure model reliability and uptime
- Automate ML workflows
- Maintain ML infrastructure

## Required Skills
- ML frameworks (TensorFlow, PyTorch, scikit-learn)
- MLOps tools and practices
- Cloud platforms (AWS, GCP, Azure)
- Programming (Python, Scala, Java)
- Distributed computing (Spark, Ray)
- Containerization (Docker, Kubernetes)
- Data pipeline development
- Model optimization techniques
- Software engineering best practices
- Version control and CI/CD

## Reporting Structure
- Reports to: ML Engineering Manager or Head of AI
- Direct Reports: None

## Success Metrics
- Model deployment frequency
- Model performance in production
- System reliability and uptime
- Pipeline efficiency
- Model serving latency
- Resource utilization
- Experiment velocity
- Code quality metrics
- Automation coverage
