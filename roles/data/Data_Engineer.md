# Data Engineer

## Overview
The Data Engineer builds and maintains data infrastructure, pipelines, and systems that enable data collection, storage, processing, and analysis at scale.

## Key Responsibilities
- Design and build data pipelines
- Maintain data warehouse infrastructure
- Implement ETL/ELT processes
- Ensure data quality and reliability
- Optimize data storage and retrieval
- Build real-time data streaming systems
- Collaborate with data scientists and analysts
- Monitor data system performance
- Implement data governance
- Document data architectures

## Required Skills
- Data pipeline development
- SQL and NoSQL databases
- ETL/ELT tools
- Cloud platforms (AWS, GCP, Azure)
- Programming (Python, Scala, Java)
- Big data technologies (Spark, Hadoop)
- Data modeling
- Stream processing
- Version control
- Problem-solving

## Reporting Structure
- Reports to: Data Engineering Manager or Head of Data
- Direct Reports: None

## Success Metrics
- Pipeline reliability and uptime
- Data processing efficiency
- Data quality metrics
- System performance
- Cost optimization
- Pipeline development speed
- Data availability SLAs
- Documentation quality
- Incident resolution time
